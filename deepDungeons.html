Deep Dungeons

 

https://github.com/CaiMills/Deep-Dungeons History Behind the Game

I created this game for a university module during my first year, and it was all done by me. I was responsible for choosing the project's theme and gameplay, as long as it was a 2D project and made in Unity.

Tools Used

• Unity
• C#
• Aseprite

About the Game

The game involves battling enemies while trying to get out of the dungeon. In order to accomplish this, the player has access to three different melee weapons and dodge maneuvers that can help prevent damage.

 

The Features I Worked On 
    • Movement Mechanics
    • Level Design
    • UI design     
    • Melee Attack System
    • Enemy AI 
    • Dodge Roll
    • Different Weapon Types
    • Player Damage and Healing
